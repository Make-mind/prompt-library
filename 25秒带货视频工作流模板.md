# 25秒带货视频工作流模板（完整优化版）

---

## 固定提问模板

根据我上传的产品图片，帮我为一个【总时长】左右的【平台/类型】带货视频，拆分成适合 AI 生成的多段画板提示词和视频提示词。

我的生成限制是：生视频软件一次最长只能生成【单段最长时长】秒，所以请帮我把完整视频拆成多段，每段都不超过【单段最长时长】秒。

我的产品是：【产品名称】
产品图中产品特点是：【产品外观、颜色、结构、材质、功能卖点】
视频需求是：【简单剧情】
视频风格是：【抖音带货短视频 / 小红书种草 / 电影级广告 / 家居生活方式 / 治愈风】
主角设定是：【狗狗 / 猫咪 / 人物；品种、颜色、性格】
场景设定是：【客厅 / 卧室 / 户外 / 摄影棚；光线、家具、色调】
核心卖点是：【卖点1、卖点2、卖点3、卖点4】

请你严格按照以下结构回答：

1. 先给我完整视频的分段方案（每段时间、主要内容、核心卖点）
2. 再给我每一段的画板提示词框架结构（主题/剧情、主角设定、场景、电影类型、镜头语言、光影、色调、构图、分镜数量、输出格式）
3. 每一段都要给我一段"画板生成提示词"，包含顶部项目标题栏、左侧主角设定、中间场景、右侧Storyboard & Shots、底部信息卡片区、6-8个分镜缩略图、全中文标注、图文混排，不是普通海报，不是单张产品图
4. 每一段都要给我一段"视频分镜提示词"，包含时长、画幅、主体、动作、场景、光影、运镜、风格、画质、每个时间点发生什么、每个时间点的口播文案、每个时间点展示的卖点
5. 每一段都要给我"负面提示词"
6. 最后给我"主体一致性约束词"
7. 最后再帮我总结一段给生图软件用的话术
8. 再总结一段给生视频软件用的话术

请严格按照这个结构输出，不要直接只给一整段提示词。要求每段视频都必须保持同一个产品主体、同一个主角、同一个场景、同一种光影、同一种色调和同一种画面风格。

## ⚠️ 关键优化约束（每段必须追加）

### 一、画板防动画化

> 严格使用真实摄影风格生成画板。不要动画风格、不要卡通风格、不要3D渲染风格、不要插画风格、不要概念图风格、不要C4D风格、不要Blender风格。画板必须是真人实拍电影级分镜板（Live-Action Cinematic Storyboard），画面质感类似真实相机实拍，不是手绘图。所有分镜缩略图必须是真实摄影质感，不是手绘草图、不是3D渲染预览。

### 二、产品一致性增强

> 所有段落中的产品必须是严格参照上传产品图生成的。产品外观、颜色、RGB色值、形状、结构、长宽比例、材质纹理、logo位置、缝线位置、五金件颜色和位置，必须与产品图完全一致。产品在画面中不能变形、扭曲、拉伸、压扁、变色、变色温、丢失部件、增加部件。帐篷的三角角度、帐篷布垂坠弧度、木杆粗细、绑带位置、垫子厚度、绒球大小，每一帧每一张都必须与产品图100%一致。如画面与产品图出现偏差，优先以产品图为准，重新生成。

### 三、物理真实性增强

> 严格遵循真实世界物理规律。所有物体必须具有真实重量感和惯性。产品被扔出或放下时，必须以正常抛物线下落，落地速度符合重力加速度9.8m/s²的表现。落地瞬间必须出现自然的布料抖动、回弹和褶皱变化。落地后所有部件必须立即静止，不能漂浮、不能缓慢下落、不能轻飘飘浮动、不能出现慢动作效果。宠物动作也必须符合真实生物运动规律：狗狗跑动有重心转移和步态交替，钻进产品时身体有自然的弯曲和挤压，睡觉时呼吸起伏自然。画面速度必须是正常速度（1x），不要慢动作、不要升格、不要延时效果。所有布料、软垫、填充物的物理碰撞和形变必须自然。不要任何违反物理常识的画面表现。

## 固定分段结构

### 第1段：0–9秒
作用：开场吸引 + 产品出场 + 主角被吸引
内容：产品进入画面、产品完整亮相、宠物看到产品、宠物靠近产品

### 第2段：9–17秒
作用：宠物使用 + 核心卖点展示
内容：宠物进入产品、和产品互动、展示结构、材质、功能细节

### 第3段：17–25秒
作用：舒适结果 + 情绪收口 + 带货转化
内容：宠物在产品里休息、睡觉、产品融入家居场景、最后留商品卡位置

---

## 每一段执行流程

1. 把产品图和需求发给 Claude → Claude 拆出画板提示词框架 + 画板生成提示词 + 视频分镜提示词 + 负面提示词 + 一致性约束词
2. 去生图软件 → 上传产品图 → 粘贴画板生成提示词 + 画板防动画化约束 → 生成画板
3. 去生视频软件 → 上传产品图 + 画板图 → 粘贴视频分镜提示词 + 负面提示词 + 产品一致性约束 + 物理真实性约束 + 主体一致性约束词 → 生成视频
4. 三段全部生成完 → 剪辑拼接成完整 25 秒视频

## 主体一致性约束词（每段都必须加）

### 产品一致性
所有段落中的产品必须是同一个产品主体，严格参考上传的产品图。产品外观、颜色、结构、比例、材质、位置关系保持一致。不得变形、不得塌陷、不得扭曲、不得变色、不得丢失关键结构。

### 主角一致性
所有段落中的宠物主角必须保持一致，同一只狗狗，品种、毛色、体型、毛发质感、性格状态保持统一。不得变成其他品种，不得忽大忽小，不得出现多余头部、多余腿、多余尾巴，不得穿模。

### 场景一致性
所有段落保持同一个场景，同一套家居环境，同一种光线方向，同一种色调氛围。不得突然切换场景。场景风格、家具、地面、墙面、窗帘和光影保持连续。

### 风格一致性
所有段落保持同一种抖音带货短视频风格，真实摄影质感，温馨治愈，镜头语言自然，画面清晰，不要卡通风、插画风、过度电影特效或夸张转场。

## 套用案例：宠物帐篷

根据我上传的产品图片，帮我为一个25秒左右的抖音带货视频，拆分成适合 AI 生成的多段画板提示词和视频提示词。

我的生成限制是：生视频软件一次最长只能生成10秒，所以请帮我把完整视频拆成3段，每段都不超过10秒。

我的产品是：宠物帐篷狗窝。
产品图中产品特点是：三角金字塔帐篷造型，米白色帐篷布，浅色实木支架，顶部浅蓝色小帽，顶端红色圆形绒球，正面半开放门帘，两侧浅蓝色绑带，内部浅蓝色软垫，前方蓝色与棕色竖条纹长条枕垫，整体奶油风，适合小型犬休息。
视频需求是：第一段帐篷被扔进画面，狗狗看到后跑向帐篷；第二段狗狗钻进帐篷，展示半开放入口、稳定结构和产品细节；第三段狗狗在帐篷里睡觉，展示软垫舒适、安心感和带货转化。
视频风格是：抖音带货短视频，电影级画板，真实家居摄影感，温馨治愈，干净明亮。
主角设定是：一只白色小型犬，类似比熊犬或马尔济斯犬，毛发蓬松，活泼好奇，最后在帐篷里安心睡觉。
场景设定是：温馨奶油风客厅，浅色木地板，白色纱帘，原木家具，浅色沙发，绿植，自然光从左侧窗户照入。
核心卖点是：颜值高、狗狗主动钻入、半开放入口有安全感、软垫舒服、结构稳定、布料有质感、适合放在客厅。

请你严格按照结构输出。
