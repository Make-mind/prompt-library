# prompt-library — AIGC 提示词库（团队版）

## 项目定位

面向宠物行业 AIGC 从业者的提示词管理平台。从最初单机 HTML 页面，逐步演进为支持团队协作的在线网站。

**在线地址**：https://make-mind.github.io/prompt-library/
**本地运行**：双击 `启动.bat` → 浏览器打开 `http://localhost:8765`

---

## 功能模块

### 📚 提示词库（核心）
- 八维结构化管理：主体 / 动作 / 环境 / 光线 / 构图 / 风格 / 画质 / 负面词
- 类型分类：🖼️ 生图 / 🎬 生视频
- 侧栏筛选：按类型、工具、场景、评分、标签筛选
- 搜索、排序（最新/最早/评分/使用次数）
- 网格/列表双视图
- 卡片展示：标题、工具标签、场景、缩略图条、评分、使用次数
- 右侧详情面板：完整提示词 + 八维拆解 + 附件画廊 + 关联生成记录
- 新建/编辑弹窗：结构化输入（八维表单）或自由输入 + AI 智能解析
- AI 解析（豆包 API）：只做拆分归类，不修改原文
- 一键复制完整提示词、标记使用、导出/导入 JSON

### 🧩 模板引擎
- 填空式生成提示词：宠物类型 → 品种 → 场景 → 风格 → 光线 → 构图 → 画质 → 工具
- 5 套快速预设：宠物产品 / 痛点展示 / 爆款带货 / 宠物肖像 / 使用场景
- 实时预览完整提示词 + 八维结构拆解
- 一键保存到提示词库 / 复制
- 根据工具自动推断类型（Sora2/Seedance → 生视频）

### 📊 生成日志 & 成本追踪
- 记录每次 AI 生成：工具、花费、提示词、结果、生成/可用张数
- 统计卡片：累计花费、成功率、可用素材数、本月花费、平均成本
- 按工具/结果筛选
- 关联提示词库已有条目
- 导出 CSV（可用 Excel 分析）

### 🔀 工作流画布
- ComfyUI 风格的节点连线编辑器
- 4 种节点：🎯 开始设定 / 📝 提示词 / 🖼️ AI 生图 / 🎬 AI 生视频 / ✅ 审核记录
- 拖拽添加节点、连线建立流程
- 画布平移 + Ctrl+滚轮缩放
- 多工作流支持（新建/切换/重命名/删除）
- 提示词节点可从库中选择已有条目
- 节点数据自动保存

### 📝 备忘录
- 自由书写笔记
- 左侧列表 + 右侧编辑器
- 输入即自动保存（500ms 防抖）
- 显示更新时间和字数

### 📎 附件（图片/视频）
- **云端模式**（登录后）：上传到 Supabase Storage，团队共享
- **本地模式**（未登录）：存到本地文件夹（File System Access API）
- 卡片缩略图条、详情面板画廊、全屏查看器（Lightbox）
- 支持格式：JPG/PNG/GIF/WebP/AVIF/SVG + MP4/WebM/MOV/AVI
- 限制：图片 ≤10MB、视频 ≤100MB、每提示词最多 20 个文件
- 缩略图自动生成（图片 Canvas 缩放、视频取首帧）

### 🔑 团队协作
- 邮箱 + 密码登录（Supabase Auth）
- 首次登录自动注册
- 登录后数据自动同步到云端（Postgres 数据库）
- 附件上传到 Supabase Storage
- 所有登录用户共享同一份数据
- 未登录时使用 localStorage 本地存储

## 技术架构

### 前端
- 纯 HTML/CSS/JS，单文件（~3000 行，~180KB）
- 零框架、零构建工具
- CSS 变量主题系统、响应式适配
- 精致工具风 UI 设计

### 后端（Supabase）
- **数据库**：Postgres（4 张表：prompts / generation_logs / workflows / memos）
- **认证**：邮箱 + 密码登录
- **存储**：附件上传到 Supabase Storage（公开 bucket）
- **安全**：Row Level Security（可切换个人隔离 / 团队共享）

### 部署
- 前端：GitHub Pages（`make-mind.github.io/prompt-library/`）
- 后端：Supabase 托管（免费额度）
- 开发环境：Python HTTP Server（`python -m http.server 8765`）

### 第三方服务
- **Supabase**：数据库 + 认证 + 文件存储（免费方案）
- **豆包 API**（ep-20260607231451-mjjpc）：AI 智能解析提示词
- **GitHub Pages**：静态网站托管（免费）

## 数据存储

### localStorage
| Key | 内容 |
|-----|------|
| `prompt_library_v2` | 提示词数据（JSON 数组） |
| `generation_logs_v1` | 生成日志（JSON 数组） |
| `workflow_canvas_v1` | 工作流画布（JSON 数组） |
| `memo_notes_v1` | 备忘录（JSON 数组） |
| `_sb_token` | Supabase 登录令牌 |
| `_sb_user` | 当前登录用户信息 |

### IndexedDB
| 数据库 | 用途 |
|--------|------|
| `prompt_media_v1` | 存储目录句柄（`_meta` store） |

### Supabase（云端）
- Postgres 表：prompts / generation_logs / workflows / memos
- Storage bucket：attachments（公开）
