# prompt-library — AIGC 提示词库

## 项目定位
网页版提示词管理工具，三合一：提示词库 + 模板引擎 + 生成日志/成本追踪。用于保存、检索、快速生成 AI 生图/生视频的有效提示词，支持团队协作。

## 使用方式
浏览器打开 `index.html`，三个模块通过顶部导航切换：

### 📚 提示词库
- 新建提示词 — 结构化八维表单或自由输入 + AI 解析
- 浏览/搜索 — 按工具、场景、标签、评分筛选
- 查看详情 — 点击卡片打开右侧详情面板（含关联生成记录）
- 复制/使用 — 一键复制完整提示词，记录使用次数
- 导出/导入 — JSON 格式（含提示词+日志），团队间共享

### 🧩 模板引擎
- 填空式生成：选宠物类型→品种→场景→风格→光线→构图→画质→工具
- 5 套快速预设：宠物产品 / 痛点展示 / 爆款带货 / 宠物肖像 / 使用场景
- 实时预览完整提示词 + 八维结构拆解
- 一键保存到提示词库 / 复制

### 📊 生成日志 & 成本追踪
- 记录每次 AI 生成的工具、花费、提示词、效果
- 统计卡片：累计花费、成功率、可用素材数、本月花费、平均成本/素材
- 按工具/结果筛选
- 关联提示词库中的已有提示词
- 导出 CSV（方便在 Excel 中分析）
- 详情面板自动展示该提示词的所有历史生成记录

## 核心概念

### 八维提示词结构（来自《AIGC 辅助拍摄》文档）
1. 主体 (Subject) — 画面核心对象
2. 动作 (Action) — 主体行为/姿态
3. 环境 (Environment) — 场景/背景
4. 光线 (Lighting) — 光照条件
5. 构图 (Composition) — 视角/镜头
6. 风格 (Style) — 艺术风格
7. 画质 (Quality) — 清晰度相关
8. 负面词 (Negative) — 不想要的内容

### 工具覆盖
GPT-image2 / 即梦 / Midjourney / Nano Banana Pro / Sora2 / Omni / Seedance2.0 (PC/App)

### 场景分类
静物延申 / 使用图延申 / 多猫互动 / 图生视频 / 真人替换 / 爆款模仿 / 产品痛点 / 直播切片

## 数据存储
- localStorage 本地持久化，两个 key：
  - `prompt_library_v2` — 提示词数据
  - `generation_logs_v1` — 生成日志数据
- 导出 JSON 包含两者，导入自动合并去重

## 技术栈
- 纯 HTML/CSS/JS，单文件，零依赖
- localStorage 本地持久化
- AI 解析调用豆包 API（ep-20260607231451-mjjpc）
- JSON/CSV 导出实现团队共享与数据分析

## 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+1/2/3` | 切换到提示词库/模板引擎/生成日志 |
| `Ctrl+N` | 新建（当前页面：提示词弹窗 或 日志弹窗） |
| `Ctrl+S` | 保存（弹窗内） |
| `Ctrl+F` | 聚焦搜索（提示词库页） |
| `Esc` | 关闭弹窗/面板 |
